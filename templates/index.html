🎮 What This Tool Does

What Should I Play? is your Steam backlog sidekick. It helps you actually make a decision and play something.

  • Filter by playtime: Focus on untouched games or games you've already started
  • Sort by length: See how long games take via HowLongToBeat integration
  • Hide free-to-play: Filter out F2P games to focus on what you actually bought
  • Game picker modes: Use Spin the Wheel, Versus Mode, or Mystery Box when you need to stop overthinking it

⚠️ Note: This is a solo hobby project deployed for fun with no commercial backing. Expect slow updates and occasional downtime. Everything is free and always will be.

👨‍💻 About the Developer

I'm a Computer Science student who genuinely spends more time browsing my Steam library than actually playing it—sound familiar?

This project was born from my own struggle with "backlog paralysis." Every feature exists because I kept wishing for it:

  • Need to know which games won't eat up your entire weekend? That's the HLTB integration.
  • Only want to scroll through actual purchases? Hide the F2P clutter.
  • Can't decide what to play? I built the Wheel because I was literally doing it manually way too often. (Probably spun it more times than I've actually finished a game 😅)

When I'm not debugging Python or wrestling with CSS, I'm usually working on my own backlog (current count: dangerously high). I genuinely believe better tools = more games actually played, and I want this to help you break through that decision paralysis too.

Found a bug or have an idea? Hit the bug report button below—I read every single one. Seriously.

🔒 Privacy & Data

  • Your data is not sold: We only fetch and store your Steam library locally for this session
  • No tracking: We use optional Google Analytics to track page views only
  • Session-based: Each Steam ID has its own independent library view
  • Safe to use: We never post to your Steam account or modify anything

❓ Troubleshooting

  • Profile is private? Go to Steam Settings → Privacy and set your library to "Public"
  • No HLTB times showing? Click "Sync" under HLTB Times (may take 2-3 minutes, scans 300+ games)
  • Free games not detecting? Click "Sync" under Free Games to scan your library (may be slow)
  • Genres loading slowly? Genre sync queries Steam for each game - this takes time on large libraries
  • Invalid Steam ID? Enter your custom URL name (username) or your 17-digit Steam ID
  • Site is slow? This runs on Render's free tier. Free instances sleep after 15 min of inactivity—first request takes 30s to wake up. If traffic is high, the site may slow down or stop working temporarily. This is a free hosting limitation.
